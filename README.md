# Bean Bot

A bot for the UFO 50 Discord Server. Bean-bot is mainly managed by **phil-macrocheira** and **risuterra**.
Suggestions for additions to the bot can be sent our way via Discord to **phil\_macrocheira** or **loverisu**.

## Commands

### / cherry|gold|gift \[optional: number] \[optional: name]

* Displays cherry, gold, or gift requirements for the specified game
* Within channels that start with 01 through 50, parameters for number or name are not required and instead the channel number is used to determine the target game

### / codes \[optional: number] \[optional: name]

* Check the terminal codes for a game
* Within channels that start with 01 through 50, parameters for number or name are not required and instead the channel number is used to determine the target game

### / mods \[optional: number] \[optional: name]

* Get a link to a UFO 50 game's mods
* Within channels that start with 01 through 50, parameters for number or name are not required and instead the channel number is used to determine the target game

### / wr \[optional: number] \[optional: name] \[optional: players]

* Check the speedrun world records for a game
* Within channels that start with 01 through 50, parameters for number or name are not required and instead the channel number is used to determine the target game

### / random

* Get a random UFO 50 game suggestion

### / 50club

* See how many discord members have obtained 50 cherries based on discord roles

### / guides

* Get a link to steam guides for UFO 50

### / music

* Get a link to UFO 50 music on Bandcamp

### / wiki

* Get a link to the UFO 50 wiki

### / moddinginfo

* Get links for UFO 50 modding

### / tierlist

* Get a link to the Tier List Maker for UFO 50
